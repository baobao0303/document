# Cài đặt & Yêu cầu

## Yêu cầu hệ thống

Trước khi cài đặt **@cci-web/core**, hãy đảm bảo dự án của bạn đáp ứng các điều kiện sau:

- Node.js >= 18.x
- NPM >= 9.x hoặc Yarn >= 1.22.x
- Angular >= 19.2.0
- TypeScript >= 5.0.0
- Đã cấu hình Module Federation (nếu dự án là Micro Frontend)

---

## Cài đặt thư viện

### Sử dụng npm

```bash
npm install @cci-web/core
```
